Computational Ecology & Data Visualization

A collection of data visualizations I’ve produced for scientific publications and fun using the {ggplot2} package in the programming language R. All visualizations are the result of the code without any post-processing. Click on an image to enlarge it.

Contributions to #TidyTuesday

#TidyTuesday is a weekly data project aimed at the R ecosystem and the {tidyverse} package bundle to wrangle, explore and visualize the data set that is provided by the R4DS Online Learning Community.


Contributions to #MakeoverMonday

#MakeoverMonday is a weekly social data project aimed to help people learn how to create better, more effective visualizations.


Contributions to the #SWDchallenge

#SWDchallenge (“Storytelling with Data” challenge) is a monthly challenge where you can practice and apply data visualization and storytelling skills initiated by [Cole Nussbaumer Knaflic]-


“Evolution of a ggplot” blog series

“Evolution of a ggplot” is a blog series aimed to show the steps needed to transform a basic ggplot into a visually appealing visualization.


If you have some data and would like to get it visualized, don’t hesitate to contact me!
